# ZARISH HIS Documentation

This directory contains the main documentation for the ZARISH Health Information System.

## Structure

- **getting-started/**: Quick start guides and tutorials
- **architecture/**: System architecture and design
- **backend/**: Backend microservices documentation
- **frontend/**: Frontend microfrontend documentation
- **fhir/**: FHIR R5 implementation details
- **api/**: API specifications and guides
- **infrastructure/**: Infrastructure and deployment
- **deployment/**: Deployment guides
